# Player Status Modifier for Crimson Desert 1.0.0.1855

GitHub-ready source and release package for the PSM fix208 d72 update based on the d49 working baseline.

## Included

- `source/player-status-modifier/` - source snapshot used for this build.
- `player-status-modifier.asi` - tested ASI build.
- `player-status-modifier.ini` - active live-tested INI copied from `bin64` at packaging time.
- `config-presets/` - archived active, DMM, and master INI variants for traceability.
- `CHANGELOG.md` - Nexus/GitHub-facing changelog.
- `GITHUB_READY_MANIFEST.md` - hashes and source/build paths.

## Deployment Note

For Crimson Desert, deploy the ASI through Definitive Mod Manager. The INI may be mirrored to `bin64` when updating active settings.
